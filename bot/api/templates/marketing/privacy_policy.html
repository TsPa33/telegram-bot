Політика конфіденційності Carpot

Ця сторінка пояснює, які дані може обробляти Carpot для роботи Telegram-бота, створення сайтів, обробки заявок і комунікації з користувачами.

1. Які дані ми можемо збирати

  • Telegram ID, username та інші відкриті дані Telegram-профілю, які потрібні для ідентифікації користувача в боті.
  • Номер телефону, якщо користувач добровільно передає його для зв'язку або створення профілю.
  • Дані профілю продавця: назва бізнесу, опис, місто, адреса, послуги, товари, авто, ціни, графік роботи та контактні дані.
  • Дані із форм заявок на сайтах: ім'я, телефон, повідомлення та сторінка, з якої надіслано заявку.
  • Технічні дані, потрібні для безпечної та стабільної роботи сервісу.

2. Для чого використовуються дані

  • Створення та підтримка облікового запису користувача в Telegram-боті.
  • Формування сайту, каталогу або сторінки послуг продавця.
  • Передача заявок продавцю та комунікація з клієнтами.
  • Підтримка користувачів, уточнення замовлень і налаштування пакетів.
  • Поліпшення стабільності, безпеки та якості сервісу.

3. Сторонні сервіси

Для роботи Carpot можуть використовуватися сторонні сервіси, які обробляють дані відповідно до власних правил:

  • Telegram — для роботи бота та повідомлень.
  • Railway або інший серверний хостинг — для розміщення застосунку.
  • PostgreSQL — для зберігання даних сервісу.
  • Cloudinary — для зберігання й показу зображень, якщо вони використовуються.
  • LiqPay — для платежів, якщо оплата активована або використовується в конкретному пакеті.

4. Передача та захист даних

Ми не продаємо персональні дані. Дані можуть передаватися лише в межах надання сервісу, вимог закону або роботи зазначених технічних провайдерів. Ми застосовуємо розумні організаційні та технічні заходи для захисту інформації.

5. Звернення щодо даних

Користувач може звернутися щодо уточнення, оновлення або видалення даних, якщо це не суперечить вимогам законодавства та технічній роботі сервісу.
